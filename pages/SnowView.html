RMBL Snow View

RMBL Snow View is a binary semantic image segmentation project designed to identify snow in high-precision aerial photos. As part of a research initiative at the Rocky Mountain Biological Laboratory (RMBL), the project utilized machine learning techniques to achieve 99% accuracy in classifying snow coverage, aiding environmental research and ecological studies.

Technologies used: Python, Keras, NumPy

GitHub Repository: https://github.com/CUBW/RMBL_SnowView
