ConCave

ConCave is a convention-centric web platform developed by Team 3 (015-3) for the Spring 2025 Software Development class at CU Boulder. The platform aims to connect convention enthusiasts by offering features such as a merchandise store, discussion forums, group-finding tools, and more.

Technologies used: JavaScript, Postman, Handlebars, HTML, CSS, Bootstrap
